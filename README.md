Ишкаев Тимур 9-2-ПО-22
Smartbudget - мобильное приложение для ведения личных финансов. Решает проблемы: отсутствие контроля расходов; сложность планирования бюджета; непонимание, на что уходят деньги. Целевая аудитория: Студенты и молодые специалисты; люди с фиксированным доходом; пользователи, желающие научиться контролировать личные финансы; успешные бизнесмены.
